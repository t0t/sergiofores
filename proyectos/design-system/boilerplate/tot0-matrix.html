Principios Fundamentales

1. Minimalismo Radical

Solo 7 variables CSS controlan todo el sistema. Cada variable tiene un propósito único y específico.

2. Proporciones Áureas

Usando φ (1.618) como ratio base, todos los tamaños mantienen armonía matemática perfecta.

3. Semántica Pura

El HTML estructura el contenido. El CSS solo lo embellece. Sin clases innecesarias.

4. Responsividad Fluida

Sin media queries fijas. Todo se adapta fluidamente usando unidades relativas y container queries.
