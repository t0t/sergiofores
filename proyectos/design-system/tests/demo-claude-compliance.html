Sistema de 3 Tamaños ÚNICAMENTE

CUMPLIMIENTO CLAUDE.md: Prohibido font-size-xs, font-size-sm, text-xl, etc.

--text-large

Títulos principales (H1)

Usado exclusivamente para H1 y títulos hero

--text-medium

Subtítulos y navegación

H2, H3, enlaces de navegación

--text-small

Todo el resto del contenido

H4-H6, párrafos, listas, botones, formularios

Font-Weight Consistente

  • Body: --font-weight-normal por defecto
  • Títulos: --font-weight-medium o --font-weight-semibold
  • Emphasis: --font-weight-bold solo cuando sea estrictamente necesario

Layout Grid Optimizado

Grid preferred • Template rows: auto auto auto • NO usar 1fr que estira contenido

Grid Preferred

display: grid sobre flexbox para layouts principales

Auto Template Rows

grid-template-rows: auto auto auto (NO 1fr)

Justify/Align

justify-items: center • align-content: start

Optimizaciones Móviles Críticas

Viewport real iOS • Performance mobile • Breakpoints específicos iPhone

Viewport Real iOS

  • Safe viewport: 100svh en lugar de 100vh
  • Hero content: calc(100vw - 20px)
  • Padding responsivo: 0 8px en header móvil
  • Secciones compactas: min-height: 40vh móvil

Performance Mobile

  • Hardware acceleration: translate3d
  • Will-change cleanup: después de animación
  • Mix-blend-mode: deshabilitado móvil
  • Cursor personalizado: solo desktop

Breakpoints iPhone

  • 375px: iPhone SE y similares
  • 430px: iPhone moderno (14, 15, 16)
  • 768px: Tablet general
  • 1024px: Desktop
