Portfolio Auto Updater - Screenshots Automáticos

Módulos Python CLI

Scripts automatizados implementados en el sistema CEREBRO-DIGITAL

  • Portfolio Auto Updater

    Script Python que captura screenshots automáticos de proyectos web y actualiza el portfolio. Categorización inteligente: ai_systems, development, cognitive, art, commercial.

  • Google Connector API

    Integración completa con Google APIs: Calendar, Gmail y Drive. Autenticación OAuth, sincronización de eventos y procesamiento automático de documentos.

  • Events Manager

    Gestión inteligente de eventos y tareas. Comandos: events hoy, events pendientes, events categoria. Integración completa con Google Calendar.

  • Variables Áureas

    5 variables CSS que controlan todo el sistema. Progresión matemática φ = 1.618 para spacing perfecto y consonancia visual universal.

  • Sistema Atómico

    Clases modulares que se componen como átomos. pt0-pt5, ch/cv, sch/scv para máxima flexibilidad y reutilización sistemática.

  • Layout Maestro

    Framework CSS evolutivo que se adapta a cualquier proyecto. TOT0 Matrix simplifica el desarrollo manteniendo máxima expresividad visual.
