Ejemplos de flechas con texto en Mermaid

Ejemplo 1: Flecha con texto usando sintaxis básica

C -->|texto| D;
graph LR C -->|texto| D;

Ejemplo 2: Flecha con texto usando la sintaxis de doble guión

C --texto--> D;
graph LR C --texto--> D;

Ejemplo 3: Enlace sin flecha con texto

C ---texto--- D;
graph LR C ---texto--- D;

Ejemplo 4: Flecha punteada con texto

C -.texto.-> D;
graph LR C -.texto.-> D;

Ejemplo 5: Flecha gruesa con texto

C ==texto==> D;
graph LR C ==texto==> D;

Ejemplo 6: Ejemplo completo con varios nodos y relaciones

A --> B; B --texto1--> C; C ==texto2==> D; D -.texto3.-> E;
graph TD A --> B; B --texto1--> C; C ==texto2==> D; D -.texto3.-> E;
